# Potpourri Plots

A comprehensive collection of scientific data visualization methods implemented in Python.

## Overview

Potpourri Plots provides reusable visualization templates for common scientific data analysis tasks. The library includes ready-to-use examples for time series analysis, statistical comparisons, and geospatial visualization.

## Features

### 1. Time Series Visualization
- **File**: `demos/time_series_demo.ipynb`
- **Capabilities**:
  - Multi-panel time series plots with proper date formatting
  - Missing data rate calculation and visualization
  - Grid layout for comparative analysis
  - Customizable line styles and markers

### 2. Statistical Bar Charts
- **File**: `demos/bar_chart_demo.ipynb`
- **Capabilities**:
  - RMSE and MAE comparison charts for model evaluation
  - Multi-task performance visualization
  - Automatic color schemes for categorical data
  - Error bars and confidence intervals

### 3. Geospatial Heatmaps
- **File**: `demos/spatial_heatmap_demo.ipynb`
- **Capabilities**:
  - Cartopy-based geospatial visualization
  - Land mask application for oceanographic data
  - Multi-panel comparison of spatial patterns
  - Custom color ranges and normalization
